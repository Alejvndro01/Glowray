## 📜 About Glowray

**Glowray** is a mod designed to transform Minecraft's atmosphere by introducing bioluminescent wildlife. Built with a Vanilla+ philosophy, the mod integrates seamlessly into the original ecosystem, offering a unique visual experience—especially when paired with shaders.

## ✨ Key Features

* **Bioluminescent Fauna**: Includes creatures like the Stingray, a glowing manta ray that lights up ocean biomes.
* **Native Animation System**: Features smooth, custom implementation **without the need for heavy dependencies** like GeckoLib.
* **Modular Architecture**: Scalable code designed to streamline the creation of new creatures and features.
* **Vanilla+ Style**: Crafted to feel like a natural expansion of the base game.

## 🛠️ Requirements & Installation

* **Minecraft Version**: 1.20.1
* **Mod Loader**: Forge (Recommended 47.4.18+)

1. Download the mod's `.jar` file.
2. Ensure you have **Forge 1.20.1** installed.
3. Drop the file into your `mods` folder and launch the game.
