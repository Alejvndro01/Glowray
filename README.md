# Glowray

Glowray is a Minecraft mod focused on adding bioluminescent creatures and natural fauna to the game.

## Features

- Glowray (bioluminescent stingray)
- Custom animation system (no GeckoLib)
- Scalable and modular architecture

## Requirements

- Minecraft 1.20.1
- Forge 47.4.18

## Installation

1. Install Forge
2. Place the mod in your mods folder
3. Launch the game
